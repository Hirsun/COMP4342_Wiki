::: tip 软件工程：好的软件的基本属性

1. 可接受性：软件必须可理解有用，并且与用户使用的其他系统相兼容。
2. 可依赖性和信息安全性：软件可依赖性包括一系列特性，如可靠性、信息安全性、安全性。
   - 可依赖的软件即使在系统失效时也不应当导致物理或经济上的破坏。
   - 软件必须保证信息安全，使得恶意用户无法访问或破坏系统。
3. 效率：软件不应当浪费系统资源，例如存储和处理器周期。
4. 可维护性：软件应当能够通过演化满足客户变化的需求。

:::
